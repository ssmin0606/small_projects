### 3. what functions can be served
---

- 박스 체크 (여러 박스 동시 체크 가능)

<br>

### 4. Note
---

- `input:checked + p` :  checked된 input 다음에 바로 오는 p 선택하는 css selector
- `checkbox.checked` :  input type이 checkbox의 체크상태를 `true` 와 `false`를 이용해서 컨트롤 가능
