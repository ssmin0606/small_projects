### 4. Note

- `flex: 1` : flex-grow 적용
- `transform: translateY(수치)`: 수치만큼 y방향으로 움직이게 함
- `classList.toggle('클래스명')`:   클래스가 존재한다면 제거하고 `false`를 반환하며, 존재하지 않으면 클래스를 추가하고 `true`를 반환한다.
